# AI-Prompts

This repository stores reusable AI prompt instruction files for Zapier automations, ChatGPT workflows, and future marketing/content automations.

Prompts are version controlled so they can be safely reviewed, improved, and rolled back over time. Keeping prompt instructions in Git makes it easier to track changes, test improvements, and maintain consistency across automations and workflows.

Zapier automations can reference these prompt files or copy their instructions into AI steps. ChatGPT workflows and other content automation systems can also reuse these prompts as starting points for repeatable, consistent outputs.

> **Warning:** Do not store API keys, passwords, client secrets, or private client data in this repository.
